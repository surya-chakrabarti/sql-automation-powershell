# SQL Server Patch Automation (PowerShell)

## Overview
This project automates SQL Server installation and patching using PowerShell and dbatools.

## Features
- Automated patch installation
- Remote execution support
- Logging and error handling
- Controlled restart mechanism
- Scalable for multiple servers

## Technologies Used
- PowerShell
- dbatools
- SQL Server

## How to Run
1. Install dbatools module
2. Update parameters in script
3. Execute:
   powershell -ExecutionPolicy Bypass -File sql-patch-automation.ps1

## Use Case
Designed for enterprise environments to standardize SQL patching and reduce manual effort.
